# Homepage (Start Here) {.unnumbered}

![](images/Cover.jpg)

Welcome! This book is designed to help you learn statistics in a way that is practical, approachable, and relevant to real-world research and decision making.

------------------------------------------------------------------------

### What You’ll Learn {.unlisted .unnumbered}

By using this book, you will learn how to:

-   Understand and interpret statistical results
-   Analyze data using *jamovi*
-   Connect statistics to research and real-world decisions
-   Communicate findings clearly using APA-style reporting

------------------------------------------------------------------------

### Get Started with *jamovi* {.unlisted .unnumbered}

This book uses *jamovi*, a free and user-friendly statistical software.

-   Download *jamovi*: <https://www.jamovi.org>
-   Use the example datasets provided in each chapter
-   Follow step-by-step instructions to run analyses

*You do not need prior experience with statistics or software—this book will guide you.*

------------------------------------------------------------------------

### A Note About Learning Statistics {.unlisted .unnumbered}

Statistics can feel overwhelming at first—and that’s completely normal.\
This book is designed to help you build confidence step by step, even if you’ve struggled with math in the past.

Focus on understanding what results **mean**, not just how to calculate them.

------------------------------------------------------------------------

### About this Book {.unlisted .unnumbered}

This is an open educational resource (OER), meaning it is **free to use and continuously improved** based on feedback from students and instructors.

This book was created to:

-   Make statistics more accessible and applied

-   Emphasize interpretation over memorization

-   Integrate hands-on data analysis using *jamovi*

It is built using open-source tools and updated regularly to reflect new features, examples, and feedback.

If you notice an error or have a suggestion, please email me at linnellda\@uwstout.edu.
